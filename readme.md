# 首先更新nodejs
在windosw下，直接移除之前安裝的nodejs,在安裝新的
# 在安裝aggular/cli
在console下
```
npm install -g @angular/cli
```
# 建立webapi
建立一個目錄webapi，在目錄下執行
```
dotnet new webapi
```

# 建立angularapp
在Angular7的目錄下執行
```
ng new static-app
```
